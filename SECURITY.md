# Security Policy

## Supported Version

| Version | Supported |
|----------|-----------|
| 1.x | Yes |

---

## Reporting

Security issues should be reported privately to the repository maintainers.

Do not disclose security vulnerabilities publicly before they have been reviewed.

---

## Scope

This repository primarily contains engineering knowledge and documentation.

Any security concern related to repository content, workflows, or future tooling should be reported through the official maintenance channel.
