# Lifecycle Stages

1. Project Initiation
2. Knowledge Acquisition
3. Requirement Analysis
4. System Design
5. Development
6. Testing
7. Deployment
8. Maintenance
9. Project Closure

---

# Stage Definition

## Project Initiation

- Define objectives
- Identify stakeholders
- Define scope

## Knowledge Acquisition

- Collect stakeholder knowledge
- Validate knowledge
- Organize knowledge

## Requirement Analysis

- Analyze business needs
- Produce requirements

## System Design

- Produce engineering design

## Development

- Produce implementation artifacts

## Testing

- Validate implementation

## Deployment

- Release system

## Maintenance

- Improve system

## Project Closure

- Archive project
- Final documentation

---

# Lifecycle Rules

- Setiap tahap memiliki input dan output.
- Tahap berikutnya dimulai setelah tahap sebelumnya selesai.
- Seluruh perubahan harus terdokumentasi.
- Seluruh Artifact harus dapat ditelusuri.
