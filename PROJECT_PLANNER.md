# Project Planner

Version: 1.0
Status: Final

---

# Purpose

Project Planner bertanggung jawab menyusun rencana software engineering berdasarkan Knowledge dan Workflow yang tersedia.

Project Planner tidak menghasilkan Engineering Artifact.

---

# Principles

- Knowledge Driven
- Goal Oriented
- Sequential
- Traceable
- Standardized

---

# Responsibilities

Project Planner bertanggung jawab untuk:

- memahami tujuan Project;
- menyusun tahapan software engineering;
- menentukan urutan Task;
- menentukan Artifact yang harus dihasilkan;
- memastikan seluruh proses mengikuti Workflow.

---

# Planning Flow

Project

↓

Knowledge

↓

Workflow

↓

Tasks

↓

Artifacts

---

# Rules

- Seluruh Task harus berasal dari Workflow.
- Setiap Task harus memiliki Artifact.
- Setiap Artifact harus dapat ditelusuri ke Knowledge.
- Perencanaan harus dilakukan sebelum proses engineering dimulai.

---

# Inputs

- Project
- Knowledge
- Workflow

---

# Outputs

- Project Plan
- Task Sequence
- Artifact Sequence

---

# Constraints

Project Planner tidak:

- menjalankan Task;
- menghasilkan Artifact;
- mengubah Knowledge.
