# Repository Conventions

Version: 1.0
Status: Final

---

# Naming Convention

- Gunakan lowercase.
- Gunakan kebab-case untuk nama file dan folder.
- Gunakan bahasa Inggris.
- Hindari singkatan yang tidak umum.

---

# Documentation Convention

Seluruh dokumen harus memiliki:

- Title
- Version
- Status
- Purpose
- References
- Version History

---

# Repository Convention

- Ikuti struktur repository.
- Jangan menduplikasi Knowledge.
- Gunakan Template yang tersedia.
- Perbarui CHANGELOG apabila diperlukan.
