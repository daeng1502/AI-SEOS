# Team

Version: 1.0
Status: Final

---

# Purpose

Team mendefinisikan struktur kolaborasi Role dalam menjalankan proses software engineering.

Team tidak mendefinisikan AI Agent maupun implementasi AI.

---

# Principles

- Collaboration
- Clear Responsibility
- Traceability
- Standardized
- Reusable

---

# Standard Team Structure

- Project Manager
- Business Analyst
- System Analyst
- Software Architect
- UI/UX Designer
- Database Engineer
- Backend Engineer
- Frontend Engineer
- QA Engineer
- Technical Writer

---

# Responsibilities

Team bertanggung jawab untuk:

- melaksanakan Workflow;
- menjalankan Task sesuai Role;
- menghasilkan Engineering Artifact;
- menjaga konsistensi Knowledge.

---

# Collaboration Flow

Project Manager

Software Architect

↓

↓

Frontend Engineer

↓

QA Engineer

↓

---

# Rules

- Setiap Role memiliki tanggung jawab yang jelas.
- Kolaborasi mengikuti Workflow.
- Seluruh Artifact harus memiliki Role utama.
- Setiap Role menggunakan Skill yang sesuai.

---

# Inputs

- Project
- Workflow
- Tasks

---

# Outputs

- Engineering Artifacts

---

# Constraints

Team tidak bergantung pada AI Provider maupun AI Orchestrator.
