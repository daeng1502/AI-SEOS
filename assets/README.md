# Assets

Folder ini berisi aset pendukung yang digunakan oleh AI-SEOS.

## Purpose

Assets digunakan untuk mendukung dokumentasi, presentasi, dan identitas repository, namun tidak menjadi bagian dari Engineering Knowledge maupun Workflow AI-SEOS.

## Typical Contents

- Repository logo
- Diagrams
- Icons
- Images
- Banners
- Illustrations

## Rules

- Gunakan format aset yang umum digunakan (PNG, SVG, JPG, PDF).
- Berikan nama file yang deskriptif.
- Hindari duplikasi aset.
- Simpan aset sesuai kategori apabila jumlah aset bertambah.

## Notes

Folder ini tidak digunakan sebagai sumber Knowledge maupun Configuration AI-SEOS.
