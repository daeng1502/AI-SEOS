# Stakeholder Interview

## Project

Student Information System

---

## Stakeholders

### Administrator

Goals

- Mengelola data siswa.
- Mengelola data guru.
- Mengelola kelas.
- Mengelola tahun ajaran.

Problems

- Data tersebar di banyak file.
- Sulit membuat laporan.

---

### Teacher

Goals

- Mengelola nilai siswa.
- Melihat data kelas.
- Mengelola absensi.

Problems

- Penginputan nilai memakan waktu.
- Rekap nilai dilakukan secara manual.

---

### Student

Goals

- Melihat nilai.
- Melihat jadwal.
- Melihat absensi.

Problems

- Informasi akademik sulit diakses.
