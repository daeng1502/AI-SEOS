# Examples

Folder ini berisi contoh Project yang digunakan untuk menunjukkan bagaimana AI-SEOS menerima Knowledge dari stakeholder dan menghasilkan Engineering Artifact.

Seluruh contoh pada folder ini menggunakan standar, template, workflow, dan playbook yang telah didefinisikan oleh AI-SEOS.

Contoh Project tidak dimaksudkan sebagai implementasi akhir, tetapi sebagai referensi penggunaan AI-SEOS.

## Available Examples

- Library Management System
- Student Information System

Setiap contoh Project minimal memiliki:

- Stakeholder Interview
- Business Rules
- Functional Requirements
- Non-Functional Requirements
- Project Metadata

Contoh tersebut menjadi input awal yang akan diproses oleh AI Orchestrator menggunakan AI-SEOS.
