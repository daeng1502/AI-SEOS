# Definition of Done

Version: 1.0
Status: Final

---

# Purpose

Definition of Done (DoD) mendefinisikan kondisi yang harus dipenuhi agar suatu Task atau Engineering Artifact dinyatakan selesai.

---

# Principles

- Clear
- Objective
- Measurable
- Repeatable
- Traceable

---

# Definition of Done

Sebuah Task dinyatakan selesai apabila:

- seluruh Requirement telah dipenuhi;
- seluruh Deliverable telah dihasilkan;
- Review telah selesai;
- Validation telah selesai;
- Acceptance Criteria telah terpenuhi;
- seluruh perubahan telah terdokumentasi.

---

# Artifact Definition of Done

Setiap Artifact harus:

- Complete
- Correct
- Consistent
- Traceable
- Versioned
- Approved

---

# Task Definition of Done

Setiap Task harus:

- memiliki status Completed;
- menghasilkan seluruh output yang diwajibkan;
- tidak memiliki blocker;
- telah direview.

---

# Completion Checklist

- Requirement Completed
- Deliverable Completed
- Review Completed
- Validation Completed
- Documentation Updated
- Approval Granted

---

# Deliverables

- Completed Task
- Approved Artifact
