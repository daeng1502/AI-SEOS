# Code of Conduct

## Purpose

AI-SEOS is an engineering knowledge repository.

All contributions should prioritize:

- professionalism;
- technical accuracy;
- consistency;
- traceability;
- maintainability.

---

## Expected Behavior

Contributors should:

- follow repository standards;
- document every significant change;
- maintain consistency;
- review contributions before submission;
- respect existing governance.

---

## Unacceptable Behavior

- undocumented changes;
- inconsistent documentation;
- breaking repository standards;
- introducing ambiguous knowledge.

---

## Enforcement

Repository maintainers may request revisions or reject contributions that do not comply with repository standards.
