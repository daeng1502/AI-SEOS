# Context

Version: 1.0
Status: Final

---

# Purpose

Context merupakan kumpulan Knowledge yang telah dipilih dan disusun agar relevan dengan tugas yang sedang dikerjakan.

Context memastikan AI hanya menggunakan informasi yang diperlukan untuk menghasilkan Artifact.

# Context Rules

- Context harus berasal dari Knowledge yang valid.
- Context hanya berisi informasi yang relevan.
- Context tidak boleh mengandung informasi yang bertentangan.
- Context harus dapat ditelusuri kembali ke sumbernya.
