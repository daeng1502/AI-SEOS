# Skills

Version: 1.0
Status: Final

---

# Purpose

Skills mendefinisikan kemampuan yang diperlukan untuk menjalankan suatu Role dalam menghasilkan Engineering Artifact.

Skill merupakan kemampuan engineering, bukan kemampuan AI Model.

---

# Principles

- Reusable
- Measurable
- Independent
- Traceable
- Domain Agnostic

---

# Skill Categories

- Requirement Analysis
- Business Process Analysis
- Software Architecture
- UML Modeling
- Database Design
- UI/UX Design
- API Design
- Backend Development
- Frontend Development
- Testing
- Documentation
- Review

---

# Skill Definition

Setiap Skill harus memiliki:

- Purpose
- Inputs
- Outputs
- Related Roles
- Produced Artifacts

---

# Skill Rules

- Satu Role dapat memiliki banyak Skill.
- Satu Skill dapat digunakan oleh banyak Role.
- Skill dapat digunakan kembali pada berbagai jenis proyek.
- Skill tidak bergantung pada AI Provider maupun AI Orchestrator.

---

# Inputs

- Knowledge
- Context
- Task

---

# Outputs

- Engineering Artifact

---

# Constraints

Skill hanya mendefinisikan kemampuan software engineering.

Skill tidak mendefinisikan implementasi AI.
