# AI-SEOS

> AI Software Engineering Operating System

AI-SEOS adalah sebuah Operating System berbasis repository yang menyediakan aturan, pengetahuan, standar, workflow, dan artefak software engineering agar AI Agent dapat mengerjakan proyek perangkat lunak secara konsisten.

AI-SEOS tidak menjalankan AI.

AI-SEOS menjadi sumber kebenaran (Source of Truth) yang digunakan oleh AI Orchestrator seperti Antigravity, 9Router, atau sistem sejenis.

---

# Purpose

AI-SEOS bertujuan untuk menstandarkan proses software engineering sehingga AI dapat:

- memahami kebutuhan stakeholder;
- menyusun rencana proyek;
- menghasilkan artefak software engineering;
- menjaga konsistensi pengetahuan;
- mengikuti standar engineering yang sama pada setiap proyek.

---

# Scope

AI-SEOS mendefinisikan:

- Project Lifecycle
- Engineering Standards
